# Space Cat Food Catcher 6

Browser arcade про кота-пилота, который ловит космическую еду и избегает бомб.

## Как запустить

- Откройте `game/space-cat-food-catcher-6/index.html` в браузере;
- или поднимите любой локальный static server в корне проекта.

## Управление

- **ПК:** мышь, A/D, стрелки, W/S;
- **Мобильные:** drag/swipe по полю;
- **Esc / P:** пауза;
- **R:** рестарт.

## Особенности

- canvas-based gameplay;
- меню, настройки, gameplay, pause, result;
- RU/EN переключение;
- score, lives, combo, timer;
- 15-minute win state;
- no external CDN/fonts/images.

## Аудио

Локальные аудиофайлы из manifest не были добавлены. Вместо этого используется graceful no-asset fallback: процедурные короткие звуки через Web Audio, а при недоступности аудио — тихий mute-friendly режим.

## Примечания по стилю

Визуальный тон: neon space-kitchen, readable mobile UI, food/bomb distinction by silhouette.
